Симулятор рулетки.

=============================================================Подготовка к использованию=============================================================
1) Убедиться в наличии Java 8.
2) Распаковать все файлы в архиве в отдельную папку, где будете хранить программу.
3) Следовать инструкциям, приведенным ниже.

=============================================================Как использовать=============================================================
1) Запускаете CaseSimulator.jar
2) Выбираете нужную конфигурацию 
(Choose case name - в выпадающем меню выбираете название кейса, которого хотите открыть.
Choose case render mode - в выпадающем меню выберет режим отрисовки дропа (OFT - рулетка, PRT - просто отрисовывается дроп).
Count test iteration - количество тестовых прогонов функции рандома (требуется для проверки точности выпадения процентов). Результат теста выводится в файл "RandomTest.txt"
Random key - текстовая строка, на основе которой генерируется ключ. Одинаковые строки дадут одинаковый выпадающий лут.
Start!  - нажмите, чтобы начать.)
3) Основное окно должно открыться. Нажимайте пробел, чтобы остановить рулетку (выпал лут) и начать заново.

=============================================================Как создать свой кейс=============================================================
1) В папке "resource\config" создайте папку. Название папки будет отображаться как название кейса.
2) В созданной папке создайте файл "itemdrop.json".
3) Заполните файл по примеру:

{ //обязательно

//новый item. "gauss" - название item'а. "drop_percent" - число, отражающее шанс дропа. "texture" - путь до файла с текстурой.
  "gauss": {
    "drop_percent" : 4,
    "texture": "resource\\texture\\items\\iskluchitelniy\\gauss.png"
  },
//после объявления item'а в квадратных скобках ставить запятую. Это не относится к последнему item'у

//новый item. "tank" - название item'а. "drop_percent" - число, отражающее шанс дропа. "texture" - путь до файла с текстурой.
  "tank": {
    "drop_percent" : 4,
    "texture": "resource\\texture\\items\\iskluchitelniy\\tank.png"
  }
//после объявления item'а в квадратных скобках ставить запятую. Это не относится к последнему item'у

} //обязательно

В случае непонимания как это делать посмотрите пример уже заполненных конфигов для существующих кейсов и устройство формата JSON.
4) drop_percent отражает шанс дропа вещи. Шанс дропа вычисляется как: 1 / (сумма всех drop_percent'ов) * (drop_percent текущего предмета).
Отнеситесь к этому внимательно.
5) Добавьте все текстуры, указанные в конфигах предметов (сторона текстуры обязательно должна равняться какой-либо из степеней двойки, т.е. 2, 4, 8, 16, 32, 64 и тд).
6) Кейс готов.
